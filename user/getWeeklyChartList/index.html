    getWeeklyChartList

    Get the start and end date of every week since 13th Feburary 2005 12:00:00 UTC.

    No authentication required.

    Why the strange name?

    Originally this method was meant to get a list of available charts for a user, expressed as date ranges which could be sent to the chart services.

    Parameters¶

    Method Type Default Required Description
    api_key token none True A Last.fm API key.

    Responses¶

    • 8 : Operation failed - Something else went wrong

    • 16 : There was a temporary error processing your request. Please try again
