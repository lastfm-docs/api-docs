    getFriends¶

    Get a list of the user's friends (follow each other) on Last.fm.

    No authentication required.

    Parameters¶

    Method Type Default Optional Description
    user string none The last.fm username to fetch the friends of.
    recenttracks boolean 0 Whether or not to include information about friends' recent listening in the response.
    limit number 50 The number of results to fetch per page. Defaults to 50.
    page number 1 The page number to fetch. Defaults to first page.
    api_key token none A Last.fm API key.

    Responses¶

    Errors:

    • 6 : Invalid parameters - Your request was either missing a required parameter, the parameter was not found, or had 0 results
    • 8 : Operation failed - Something else went wrong
    • 9 : Invalid session key - Please re-authenticate
    • 10 : Invalid API key - You must be granted a valid key by last.fm
    • 11 : Service Offline - This service is temporarily offline. Try again later.
    • 13 : Invalid method signature supplied
    • 16 : There was a temporary error processing your request. Please try again
    • 26 : Suspended API key - Access for your account has been suspended, please contact Last.fm
    • 29 : Rate limit exceeded - Your IP has made too many requests in a short period
